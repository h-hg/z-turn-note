# vimtex

## 快捷键

|命令|解释|备注|
|:-:|:-:|:-:|
|`ll`|开始监听`.tex`的变化，编译并打开预览|第二次 `ll` 停止监听|
|`lv`|同步预览器||
|`lc`|清理编译生成的中间文件||
