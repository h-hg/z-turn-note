# Frp

Frp 是一个高性能的内网穿透软件。分为 frpc（客户端）和 frps（服务端）。

服务端指的是具有公网端口的服务器，需要使用 `frps` 和配置 `frps.ini`

客户端指的是在内网中提供服务的服务器，需要使用 `frpc` 和配置 `frpc.ini`

可以配置的话，可以参考官网的中文文档。
