# Ranger

重命名

按键

|:-:|:-:|
|`cw`|重命名（含后缀名）|
|`I`|重命名，光标在最前|
|`A`|重命名，光标在最后（含后缀名）|
|`a`|重命名，光标在最后（不含后缀名）|

[参考](http://www.huangpan.net/posts/ji-ke/2019-08-21-ranger.html)

## 配置

```bash
ranger --copy-config=all
```
