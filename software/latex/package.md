# 宏包

这里推荐一些宏包

## 数学

- amsmath：公式
- siunitx：单位

## 表格

- multirow：单元格处理
- makecell：单元格处理
- langtable：长表格
- xtab：常表格
- array：表列格式

## 图片

- TikZ：绘图
- forest：画树
- graphicx：插图

## PDF

- hyperref：超链接
- bookmark：书签
- pdfcomment：注释

## 代码

- listings：提高高亮
- clrscode：算法导论伪代码的宏包
- algorithm2e
- algorithmicx

## 化学

- chemformula：化学式，以前用 `mhchem`
